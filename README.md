1. GIỚI THIỆU
Trang web giới thiệu cá nhân được xây dựng bằng HTML, CSS và JavaScript thuần. Đây là dự án phù hợp cho người mới học lập trình web muốn tạo profile cá nhân cho riêng mình.

Trang web có các tính năng cơ bản:
- Giao diện rõ ràng
- Menu điều hướng mượt mà
- Form liên hệ đơn giản
- Hiển thị thông tin cá nhân
